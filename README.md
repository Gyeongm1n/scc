# scc_flutter_app

A new Flutter project.

## Getting Started

1.https://github.com/Gyeongm1n/ontact_test_server에서 서버코드 다운  
2.MongoDB다운로드  
3.cmd창에서 ontact_test_server폴더로 이동  
4.npm i 입력  
5.node index.js입력  
6.AndroidStudio에서 코드실행
